Classified information leaked
ACTION

Build an Alliance
ACTION
Count the number of players in the game (including yourself). Draw enough cards to give 1 card to each player, and then distribute them evenly amongst all the players. You decide who gets what.
Share the Wealth
ACTION
Gather up all the keepers on the table. Shuffle them together, then deal them out,giving the first card to yourself. Everyone will probably end up with a different number of Keepers in play than they started with.
Today’s Special
ACTION
Set your hand aside and draw 3 cards. If today is your birthday, play all 3 cards. If today is a holiday or special anniversary, play 2 of the cards. If it’s just another day, play only 1 card. Discard the remainder.
Congress changes hands
ACTION
Reset to Basic Rules. Discard all new rule cards, and leave only the basic rules in play. Do not discard the current Goal.
